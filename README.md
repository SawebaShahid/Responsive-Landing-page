# 🚀 Responsive Landing Page

A modern and responsive landing page built using **HTML**, **CSS**, and **JavaScript**. The project features a clean design, smooth animations, and a user-friendly interface that works seamlessly across desktop and mobile devices.

## ✨ Features

- 📱 Responsive layout
- 🎯 Hero section
- 💼 Services section
- 💰 Pricing section
- 🎨 Smooth animations
- 🖱️ Hover effects
- 🌟 Clean and modern UI

## 🛠️ Technologies Used

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/responsive-landing-page.git
   ```

2. 📂 Open the project folder.

3. 🌍 Run `index.html` in your browser.
